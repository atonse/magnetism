h1. Magnetism

Magnetism is an attempt at a website CMS/blog. Its currently under development and at this time is not meant for production *whatsoever*.

If you'd like to see the progress please do the following:

*NOTE:* Drop the database if you have cloned this repository already. I have moved from Clearance to Devise.

1. Clone the repo
2. bundle install
3. rake db:migrate
4. rake db:seed
5. rails s
6. Navigate to localhost:3000/admin
